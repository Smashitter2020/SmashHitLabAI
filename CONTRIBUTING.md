## Contributing to Smash Hit Lab Local Assistant
Thank you for your intrest in contributing!

This project is designed to be **community-friendly, modder-friendly, and easy to extend**.

Whether you're improving documentation, adding new tools, or enhancing retrieval accuracy - your help is welcome.

## How to Contribute
### Fork & Clone
```bash
git clone https://github.com/Smashitter2020/SmashHitLabAI
cd SmashHitLabAI
```

## Code Style
- Use Python 3.10+
- Keep scripts dependency-light
- Prefer clarity over cleverness
- Avoid adding heavy frameworks unless necessary.

### What Not to Contribute
To respect legal boundaries:

- Do not upload game assets
- Do not upload decompiled code
- Do not upload copyrighted content
- Do not include paid or propriety tools

## Need Help?
Open an issue or discussion on GitHub.

The community is friendly and happy to help modders of all skill levels.
